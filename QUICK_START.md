# 🚀 Быстрый старт

## ✅ Где добавлять прогресс и пополнять финансы

### 1️⃣ Обычные цели (часы, км, повторения и т.д.)

**Страница: `/today` (Сегодня)**

```
📍 ГДЕ НАХОДИТСЯ:
Меню → "Сегодня" (первая кнопка)

🎯 КАК ВЫГЛЯДИТ:
┌─────────────────────────────────────┐
│ Пробежать 5 км                      │
│ Прогресс: 2 / 5 км                  │
│ [▰▰▰▰▱▱▱▱▱▱] 40%                    │
│                                     │
│ ┌──────────┐  ┌──────────┐         │ ← ВОТ ЗДЕСЬ!
│ │ [число]  │  │ Сохранить │         │
│ └──────────┘  └──────────┘         │
│                                     │
│ [Заметка (необязательно)]           │
└─────────────────────────────────────┘

✍️ ЧТО ДЕЛАТЬ:
1. Введите число (например, "2.5")
2. Нажмите "Сохранить"
3. Готово! ✅
```

### 2️⃣ Финансы (накопления)

**Страница: `/finances` → кликнуть на цель**

```
📍 ГДЕ НАХОДИТСЯ:
1. Меню → "Финансы"
2. КЛИКНИТЕ на карточку нужной цели
3. Откроется страница `/finances/goals/[id]`

🎯 КАК ВЫГЛЯДИТ:
┌─────────────────────────────────────┐
│ Накопить на MacBook                 │ ← Кликните сюда!
│ Прогресс: 50 000 / 200 000 RUB      │
│ [▰▰▰▰▰▱▱▱▱▱] 25%                    │
└─────────────────────────────────────┘

После клика появится форма справа:

┌─────────────────────────────┐
│ Добавить запись             │
│                             │
│ Сумма:  [10000]             │ ← Введите сумму
│ Дата:   [сегодня]           │
│ Источник: [зарплата]        │
│ Заметка: [...]              │
│                             │
│ [Добавить]                  │ ← Нажмите
└─────────────────────────────┘

✍️ ЧТО ДЕЛАТЬ:
1. Кликните на карточку цели
2. Введите сумму (например, "+10000")
3. Нажмите "Добавить"
4. Готово! ✅
```

## 📋 Шаг за шагом

### Создать цель:
```
1. Меню → "Цели"
2. Форма вверху:
   - Название: "Пробежать 5 км"
   - Тип: QUANTITATIVE
   - Период: DAILY
   - Цель: 5
   - Единица: км
3. "Сохранить"
```

### Добавить прогресс:
```
1. Меню → "Сегодня"
2. Найдите цель
3. Введите число в поле
4. "Сохранить"
```

### Создать финансовую цель:
```
1. Меню → "Финансы"
2. Кнопка справа вверху (может быть кнопка создания)
3. Или перейдите в `/finances/goals/new`
4. Заполните форму
5. "Создать цель"
```

### Пополнить финансовую цель:
```
1. Меню → "Финансы"
2. КЛИКНИТЕ на нужную цель
3. Справа появится форма
4. Введите сумму
5. "Добавить"
```

## 💡 Подсказки

### Почему цель не появляется на странице "Сегодня"?

Проверьте:
- ✅ Дата начала уже наступила?
- ✅ Цель не архивирована?
- ✅ Дата окончания не прошла?

### Где форма пополнения финансов?

**Кликните на карточку цели на странице `/finances`!**

Подсказка на странице:
```
💡 Кликните на цель, чтобы пополнить
```

### Как переключить тему?

```
Меню → "Настройки" → Theme → выберите Light/Dark/System
```
